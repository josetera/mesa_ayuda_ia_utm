# Sistema Inteligente para Mesa de Ayuda TI Universitaria

## Integrantes
- José Manuel Faundes
- Miguel Ángel Ortiz

## Descripción

Sistema inteligente basado en IA para la automatización de la mesa de ayuda TI de la Universidad Tecnológica Metropolitana (UTM).

El proyecto utiliza:

- LangChain
- RAG (Retrieval Augmented Generation)
- Memoria conversacional
- Observabilidad y monitoreo
- Dashboard de métricas
